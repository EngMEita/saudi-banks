How do I know which bank from the IBAN number?

Recognizing the IBAN number aids in assessing the legitimacy of financial transfers, especially in countries using it. Verification is through bank communication. Each bank, like Bank Albilad with an IBAN number of 15, has a unified code.

The bank code in the IBAN number

Often, the IBAN number consists of 24 digits, and each section of it indicates a specific symbol, noting that it is an international standard numbering system adopted by the European Committee for Banking Standards to facilitate the conduct of financial payments operations between all banks without difficulties or delays in the financial transfer, we explain the following code Bank in IBAN number:

  • The first digit (the first two digits) is the country code
  • While the second digit (third and fourth digit): they are the verification numbers
  • The third digit (2-4 digits): indicates the bank code
  • The fourth and last digit (15 digits or more): the account number
